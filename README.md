# -LaTeX-SZPU-Thesis-LaTeX-Template
本项目为“深圳职业技术大学 (Shenzhen Polytechnic University)”毕业设计（论文）的 LaTeX 封面模板。参考“深圳职业技术大学毕业论文格式规范-2025”word项目作为模板，部分内容还需调整，用户可根据自身情况进行调整更改。
This project is a LaTeX cover template for graduation projects (theses) at Shenzhen Polytechnic University. It is based on the “Shenzhen Polytechnic University Thesis Formatting Guidelines—2025” Word template; however, some content still requires adjustment, and users may modify it according to their specific needs.
由于作者对LaTeX编辑操作并不熟练，部分内容还需要后续学习调整，请各位见谅。
As the author is not yet fully proficient in editing with LaTeX, some parts of the content will require further study and revision. We appreciate your understanding.
